Lección 15) Querer + verbo (~고 싶어요)

P.97 Conversación

창문 : 생일 선물 뭐 받고 싶어요?
나결 : 노트북을 받고 싶어요.

동규 : 점심에 저는 라면을 먹고 싶어요. 희운 씨는요?
희운 : 저는 만두를 먹고 싶어요.

좌신 : 상원 씨 내일 어디 가고 싶어요?
상원 : 좌신 씨 집에 가고 싶어요.

건우 : 오늘 밤에 뭐 하고 싶어요?
소화 : 내일 시험을 쳐요. 그래서 일찍 자요.

민기 : 새해에는 운동을 하고 싶어요. 나결 씨는요?
나결 : 저는 공부를 열심히 하고 싶어요.

P.99 Lectura

직원 : 어서오세요. 무엇을 도와드릴까요?
동규 : 택배를 보내고 싶어요.
직원 : 어디로 보내고 싶어요?
동규 : 콜롬비아로 보내고 싶어요!
직원 : 네, 안에 뭐가 있어요?
동규 : 옷하고 화장품이요.
직원 : 모두 12kg이에요.
동규 : 가격이 얼마예요?
직원 : 잠시만요, 고객님. 가격은 총 25만원이에요.
동규 : 할인 안돼요?
직원 : 아 죄송합니다, 할인은 안됩니다.
동규 : 아 네, 알겠습니다. 콜롬비아까지 얼마나 걸려요?
직원 : 일주일 정도 걸려요.
동규 : 네, 감사합니다. 여기 카드 있어요.
직원 : 네, 25만원이에요. 영수증 여기 있어요, 감사합니다.
동규 : 안녕히 계세요.
직원 : 안녕히 가세요.
