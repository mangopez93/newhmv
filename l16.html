Lección 16) Partícula -도, -의(También, De)

P.103 Conversación

재석 : 오늘도 좋은 하루 보내세요.
선우 : 재석 씨도 좋은 하루 보내세요.

민경 : 뭐 하고 있어요?
솔지 : 집에서 밥 먹고 있어요.
민경 : 저도 밥 먹고 있어요.
솔지 : 맛있게 드세요.

나결 : 스페인어를 배우고 싶어요.
창문 : 저도 스페인어를 배우고 싶어요.

P.108 Conversación

재석 : 누구(의) 우산이에요?
선우 : 제(저의) 우산이에요.

민경 : 육회는 어디 나라 음식이에요?
솔지 : 한국(의) 음식이에요.
민경 : 데킬라는 어디 나라 술이에요?
솔지 : 멕시코(의) 술이에요.

상원 : 이거 제(저의) 우산이에요.
건우 : 거짓말. 이거 동규(의) 우산이에요.
동규 : 이거 제(저의) 우산이에요.
창문 : 아니에요! 이거 제 우산이에요!
동규 : 아, 네. 여기요.

나결 : 저거 누구(의) 건물이에요?
창문 : 재석이(의) 건물이에요.
